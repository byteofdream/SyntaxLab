# SyntaxLab

SyntaxLab is a bilingual (English/Russian) learning website for popular programming languages.

The project is built with **React + TypeScript + Vite** and includes:
- A modern landing page with language cards
- EN/RU UI switcher
- Detailed language pages with explanations, topics, roadmap, guided practice, common mistakes, project ideas, and code examples
- 30+ popular language tracks in one catalog

## Tech Stack

- React 18
- TypeScript
- React Router DOM
- Vite

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```

## Available Routes

- `/en` - English home page
- `/ru` - Russian home page
- `/en/languages/:slug` - English language page
- `/ru/languages/:slug` - Russian language page

## Content Model

Language content is centralized in `src/data/tutorials.ts`.
Each track contains localized data for EN/RU and fields used by the UI (overview, use cases, roadmap, guides, snippets, etc.).

## Notes

- This repository currently does not include automated tests.
- Dist files are generated by Vite in the `dist/` folder.
